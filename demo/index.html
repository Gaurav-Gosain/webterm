webterm

An embeddable browser terminal built on xterm.js, with kitty graphics, a clipboard layer that works on insecure origins, reference-checked unicode widths, and an optional macOS-style window frame. Nothing on this page talks to a server: the terminal is driven by a small fake shell so the rendering, the graphics and the frame are all real while the far end is not.

The terminal, with the window chrome

Every control below is a live option. The frame is a separate export that imports nothing from the terminal, so the same chrome frames any content.

Kitty graphics

Placements are canvases in a DOM layer above the grid, so they can be moved, refreshed and deleted individually. Scroll the terminal after placing one: it stays anchored to the row that introduced it, which is the scrollback default.

The terminal on its own

The same package with no chrome, no graphics and no fit observer, opened into a plain container. This is what an embed costs when all you want is a terminal.

Built from dist/. Rebuild with npm run build after changing anything under src/. MIT.
